# API REST en Python con Flask

API básica desarrollada con Flask para gestión de usuarios.

## Funcionalidades
- Obtener usuarios (GET)
- Agregar usuarios (POST)

## Tecnologías
- Python
- Flask

## Endpoints

GET /usuarios  
POST /usuarios  

## Ejecución

1. Instalar dependencias:
pip install -r requirements.txt

2. Ejecutar:
python app.py
